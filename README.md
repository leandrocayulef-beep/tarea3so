Simulación de gestión de memoria virtual utilizando paginación y política de reemplazo FIFO.

Abrir una terminal en la carpeta del proyecto y ejecutar lo siguiente:
# Crear el ejecutable
g++ -std=c++11 main.cpp -o tarea3

# Ejecutar
./tarea3

# Ejemplos de inputs:

  Ingrese tamano de Memoria Fisica (MB): 64
  
  Ingrese tamano de Pagina (KB): 4
  
  Ingrese rango de tamano de procesos (MB) [min max]: 3
  
  8
